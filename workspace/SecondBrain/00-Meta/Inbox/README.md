# Inbox

> Quick-Capture Bereich für spontane Ideen, Gedanken, Notizen

## Wie funktioniert es?

### 1. Einfach reinwerfen
Wenn dir etwas einfällt:
- Neue Note hier erstellen
- Template `00-Meta/Templates/Inbox.md` verwenden
- Oder: Schnell ohne Template (YAML-Frontmatter optional)

### 2. Innerhalb von 3 Tagen verarbeiten
Jede Inbox-Note sollte:
- In den passenden Ordner verschoben werden, ODER
- In ein Projekt integriert werden, ODER
- Gelöscht werden (wenn nicht mehr relevant)

### 3. Ziel-Ordner
| Inhalttyp | Ziel |
|-----------|------|
| Projekt-Idee | `02-Projects/Ideas/` oder `02-Projects/Active/` |
| Wissen/How-To | `03-Knowledge/` |
| Tagesnotiz | `01-Daily/` |
| Archiv | `99-Archive/` |

## Warum Inbox?

**GTD-Prinzip:** "Capture first, organize later"

- Unterbricht aktuelle Arbeit nicht
- Verhindert, dass Ideen verloren gehen
- Klare Trennung: Eingang vs. Archiv

## Tools

### Manuelle Verarbeitung
1. Note öffnen
2. Inhalt reviewen
3. Entscheiden: Projekt / Wissen / Archiv / Löschen
4. Verschieben und hier verlinken

### Zukünftig: Automatisiert
- Script: `00-Meta/Scripts/process-inbox.ps1` (in Entwicklung)

---

**Regel:** Inbox ≠ Endlager. Verarbeiten innerhalb von 3 Tagen!
