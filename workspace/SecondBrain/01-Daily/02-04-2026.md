## Key Learnings

### Rust Patterns
1. **Borrow Checker**: Use standalone functions instead of &self methods when both mutable and immutable borrows needed
2. **String Concatenation**: Use `\` line continuation in format!() instead of `+` operator
3. **Trait Exports**: Must export both Struct AND Trait for methods to be usable
4. **Mutex in Async**: Use `tokio::sync::Mutex` instead of `std::sync::Mutex` for Send compatibility

### PowerShell Best Practices
- Use `${var}` syntax for variable interpolation with special chars
- Use `takeown` before `icacls` on Windows
- ASCII-only output (no emojis) for German Windows compatibility
