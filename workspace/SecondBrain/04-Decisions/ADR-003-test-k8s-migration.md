# ADR-003: K8s Migration für ECC Framework

## Kontext

Umstellung des ECC Frameworks auf Kubernetes mit Docker Containern.

## Entscheidung

Migration zu Kubernetes geplant.

**Deadline:** Ende Mai 2026

## Offene Punkte

- [ ] API Design Patterns klären
- [ ] Migrationsschritte definieren
- [ ] Testing-Strategie festlegen

## Konsequenzen

- Bessere Skalierbarkeit
- Produktionsreifere Infrastruktur
- Höherer Komplexitäts-Overhead
