### ⚠️ CRITICAL: Hardware Limitations (GTX 980M)

**Difference from Reference Setup (H100):**
| Spec | H100 (Reference) | GTX 980M (Your Setup) |
|------|------------------|----------------------|
| VRAM | 80 GB | 4 GB |
| Compute | ~1000 TFLOPS | ~5 TFLOPS |
| Time Budget | 5 min default | 5 min (same) |
| Batch Size | Large (524K tokens) | MUST REDUCE |

**Required Adjustments for GTX 980M:**
```python
# In train.py BEFORE first run:
DEVICE_BATCH_SIZE = 8        # Default: 128 - REDUCE TO 8 or 16
TOTAL_BATCH_SIZE = 2**14     # Default: 2**19 (~524K) - REDUCE to 16K
DEPTH = 4                    # Default: 8 - REDUCE for less VRAM
MAX_SEQ_LEN = 512            # In prepare.py - REDUCE from 2048
```

**Why:** GTX 980M has only 4GB VRAM vs 80GB on H100. Default config will OOM (Out of Memory).
