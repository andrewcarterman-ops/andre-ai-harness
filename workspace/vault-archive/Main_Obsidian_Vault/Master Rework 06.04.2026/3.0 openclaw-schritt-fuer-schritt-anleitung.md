## 🛡️ SCHRITT 0: Backup erstellen (UNBEDINGT MACHEN!)

### 0.1 PowerShell als Administrator öffnen

1. Drücke `Windows-Taste`
2. Tippe "PowerShell"
3. Klicke rechts auf "Windows PowerShell"
4. Wähle "Als Administrator ausführen"
5. Klicke "Ja" bei der Sicherheitswarnung

### 0.2 Zum OpenClaw-Verzeichnis wechseln

```powershell
# In das OpenClaw-Verzeichnis gehen
cd C:\Users\andre\.openclaw

# Prüfen, ob wir im richtigen Ordner sind
Get-Location
```

**Erwartete Ausgabe:** `C:\Users\andre\.openclaw`

### 0.3 Backup-Ordner erstellen

```powershell
# Backup-Ordner mit Datum erstellen
$datum = Get-Date -Format "yyyyMMdd"
$backupOrdner = "C:\Users\andre\OpenClaw-Backup-$datum"
New-Item -ItemType Directory -Path $backupOrdner -Force

# Den gesamten .openclaw-Ordner kopieren
Copy-Item -Path "C:\Users\andre\.openclaw\*" -Destination $backupOrdner -Recurse -Force

# Prüfen, ob Backup funktioniert hat
Get-ChildItem $backupOrdner
```

**Was passiert hier:**
- Ein Backup-Ordner wird erstellt: `C:\Users\andre\OpenClaw-Backup-20260406`
- Dein gesamtes `.openclaw` wird dorthin kopiert
- Falls etwas schiefgeht, können wir alles wiederherstellen

### 0.4 Git initialisieren (für Versionskontrolle)

```powershell
# Zurück zum .openclaw Ordner
cd C:\Users\andre\.openclaw

# Git initialisieren
git init

# Alle Dateien zum Git hinzufügen
git add .

# Erster Commit (Snapshot)
git commit -m "Backup vor Renovierung - $(Get-Date -Format 'yyyy-MM-dd HH:mm')"
```

**Falls Git nicht installiert ist:**
1. Lade Git von https://git-scm.com/download/win herunter
2. Installiere es (alle Standard-Einstellungen)
3. Starte PowerShell neu

## 🔄 SCHRITT 4: Sessions von JSONL zu Markdown konvertieren

### 4.1 Konverter-Script erstellen

```powershell
# Script erstellen
$converterScript = @'
# JSONL zu Markdown Konverter für OpenClaw Sessions
param(
    [string]$SourceDir = "C:\Users\andre\.openclaw\agents\main\sessions",
    [string]$TargetDir = "C:\Users\andre\.openclaw\workspace\second-brain\01-Sessions"
)

Write-Host "Konvertiere Sessions von JSONL zu Markdown..." -ForegroundColor Green

# Alle JSONL-Dateien finden (ohne .deleted und .reset)
$jsonlFiles = Get-ChildItem -Path $SourceDir -Filter "*.jsonl" | 
    Where-Object { $_.Name -notmatch "\.deleted\." -and $_.Name -notmatch "\.reset\." }

$convertedCount = 0

foreach ($file in $jsonlFiles) {
    Write-Host "Verarbeite: $($file.Name)" -ForegroundColor Cyan
    
    # Session-ID aus Dateiname extrahieren
    $sessionId = $file.BaseName
    
    # JSONL einlesen
    $lines = Get-Content $file.FullPath
    
    # Erste Zeile parsen (Metadata)
    $firstLine = $lines | Select-Object -First 1
    $metadata = $firstLine | ConvertFrom-Json -ErrorAction SilentlyContinue
    
    # Datum extrahieren
    $date = if ($metadata.timestamp) { 
        [datetime]$metadata.timestamp 
    } else { 
        $file.LastWriteTime 
    }
    
    # Alle Nachrichten sammeln
    $messages = @()
    foreach ($line in $lines) {
        try {
            $msg = $line | ConvertFrom-Json
            $messages += $msg
        } catch {
            # Überspringe ungültige Zeilen
        }
    }
    
    # Markdown-Inhalt erstellen
    $markdownLines = @()
    $markdownLines += "---"
    $markdownLines += "date: $($date.ToString('yyyy-MM-dd'))"
    $markdownLines += "time: $($date.ToString('HH:mm'))"
    $markdownLines += "session_id: $sessionId"
    $markdownLines += "agent: main"
    $markdownLines += "status: converted"
    $markdownLines += "tags: [session, converted]"
    $markdownLines += "---"
    $markdownLines += ""
    $markdownLines += "# Session $sessionId"
    $markdownLines += ""
    $markdownLines += "## Konversation"
    $markdownLines += ""
    
    foreach ($msg in $messages) {
        if ($msg.role -and $msg.content) {
            $role = switch ($msg.role) {
                "user" { "### 👤 User" }
                "assistant" { "### 🤖 Assistant" }
                "system" { "### ⚙️ System" }
                default { "### $($msg.role)" }
            }
            $markdownLines += $role
            $markdownLines += ""
            $markdownLines += $msg.content
            $markdownLines += ""
            $markdownLines += "---"
            $markdownLines += ""
        }
    }
    
    # Markdown-Datei speichern
    $outputFile = Join-Path $TargetDir "$sessionId.md"
    $markdownLines -join "`n" | Out-File -FilePath $outputFile -Encoding UTF8
    
    Write-Host "  -> Gespeichert: $outputFile" -ForegroundColor Green
    $convertedCount++
}

Write-Host "`nFertig! $convertedCount Sessions konvertiert." -ForegroundColor Green
'@

# Script speichern
$scriptPath = "C:\Users\andre\.openclaw\workspace\scripts\Convert-SessionsToMarkdown.ps1"
New-Item -ItemType Directory -Path (Split-Path $scriptPath) -Force
$converterScript | Out-File -FilePath $scriptPath -Encoding UTF8

Write-Host "Konverter-Script erstellt: $scriptPath" -ForegroundColor Green
```

### 4.2 Konverter ausführen

```powershell

```

**Was passiert hier:**
- Liest alle `.jsonl` Dateien
- Konvertiert sie zu Markdown mit YAML Frontmatter
- Speichert sie in `second-brain/01-Sessions/`
