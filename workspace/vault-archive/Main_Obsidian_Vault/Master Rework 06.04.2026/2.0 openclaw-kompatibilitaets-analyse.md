### Phase 1: Session-Konverter (Woche 1)

**Ziel:** JSONL → Markdown Konverter

```rust
// Konvertiert .jsonl Sessions zu Markdown
fn convert_session_jsonl_to_markdown(input: &Path, output: &Path) -> Result<()> {
    let content = fs::read_to_string(input)?;
    let mut messages: Vec<Message> = Vec::new();
    
    for line in content.lines() {
        if let Ok(msg) = serde_json::from_str::<Message>(line) {
            messages.push(msg);
        }
    }
    
    let markdown = format!(
        "---\nsession_id: {}\ndate: {}\nagent: {}\n---\n\n# Session\n\n{}",
        extract_session_id(input),
        extract_date(&messages),
        "main",
        messages_to_markdown(&messages)
    );
    
    atomic_write(output, &markdown)?;
    Ok(())
}
```

**Kompatibel:** ⚠️ Neuer Code, aber nutzt bestehende Sessions

### Phase 2: Second Brain Integration (Woche 2)

**Ziel:** Sessions automatisch in Second Brain syncen

```powershell
# PowerShell Script: sessions-to-second-brain.ps1
$source = "C:\Users\andre\.openclaw\agents\main\sessions\*.jsonl"
$target = "C:\Users\andre\.openclaw\workspace\second-brain\01-Sessions"

# Konvertiere und kopiere
foreach ($file in Get-ChildItem $source -Filter "*.jsonl") {
    if (-not $file.Name.Contains(".deleted.") -and -not $file.Name.Contains(".reset.")) {
        $outputName = $file.BaseName + ".md"
        # Konvertiere JSONL zu Markdown
        Convert-JsonlToMarkdown $file.FullName (Join-Path $target $outputName)
    }
}
```

**Kompatibel:** ✅ Ja, nutzt bestehende `second-brain/` Struktur
